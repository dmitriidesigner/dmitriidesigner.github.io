The three pillars of PKT

01

Media

PKT Network is a fully decentralized media network. Anyone can distribute media content with global accessibility, censorship-resistance and built-in payments.

02

Infrastructure

Participants are paid to power the network by staking the PKT cryptocurrency and operating nodes. This supports the network’s connectivity and traffic routing.

03

Decentralized domains

The network uses a blockchain-based domain name system called PKT DNS, which registers domain names that are accessible from all web browsers without requiring special software.
